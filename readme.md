
# Sine Kultura - Official Website & Admin Management System

A cloud-integrated, responsive web platform designed for **Sine Kultura** to manage digital portfolios, executive board directories, and partnership requests in real-time.

## Sine Kultura Website
https://sinekultura-web-87ee8.web.app

## 🚀 Tech Stack
* **Frontend:** HTML5, CSS3 (Mobile-First Responsive Design), JavaScript (ES6 Modules)
* **Backend-as-a-Service (BaaS):** [Firebase](https://firebase.google.com/) (Hosting, Realtime Database, Authentication)
* **Media Management:** [Cloudinary](https://cloudinary.com/) (CDN-based image and video optimization)
* **Environment:** Node.js (for Firebase CLI and deployment)

## ✨ Key Features
* **Cinematic Hero Section:** Fully responsive background video with zero-gap viewport optimization.
* **Real-time Portfolio Management:** Admin dashboard to upload, edit, and delete photo/video albums via Cloudinary API.
* **Dynamic Executive Board:** Update the organization's officer list instantly without code changes.
* **Secure Admin Dashboard:** Protected by Firebase Authentication with a custom mobile-responsive navigation bar.
* **Responsive Tables:** Horizontal scrolling and touch-optimized UI for managing membership data on the go.

## 📸 System Architecture
The system uses a serverless architecture to ensure high availability and low latency.

## 🛠️ Local Development & Deployment

### Prerequisites
* Node.js (v18 or higher)
* Firebase CLI (`npm install -g firebase-tools`)

### Setup
1. Clone the repository:
   ```bash
   git clone [https://github.com/carljen/sinekultura-website.git](https://github.com/carljen/sinekultura-website.git)
   ```
2. Initialize Firebase:
   ```bash
   firebase login
   ```
3. Deploy to production:
   ```bash
   firebase deploy
   ```
